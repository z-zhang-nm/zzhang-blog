004 对象的构造和析构

七月 7, 2020 发布在 C++, C++视频课程

1 构造函数

　　创建类的对象时，常常需要做一些初始化工作，C++提供了构造函数来进行这一工作，它是一种特殊的成员函数，不需要用户来调用它，其在创建对象时自动执行。

1.1 定义与调用

  1. 函数名与类名相同且可以有参数
  2. 没有任何返回类型的声明
  3. 一般情况下C++编译器会自动调用构造函数，某些情况下也需要手动调用

1.2 构造函数的分类

  1. 无参数构造函数，创建对象时只写变量名会调用类A a，注意变量名后不用写括号
  2. 有参数构造函数
  3. 拷贝(赋值)构造函数类A(const 类A& obj) {}

1.3 调用有参数构造函数的三种方法

Test(int a, int b) {}

  1. 括号法Test t1(1,2)，自动调用
  2. 等号法Test t2 = (3,4,5,6)，c++对等号符功能增强，会调用只有一个参数的构造函数，参数传递逗号表达式最后一个元素（逗号表达式最后一个值为整个表达式的值）
  3. 直接调用构造函数Test t3 = Test(1,2)，手动调用，会产生一个匿名对象

1.3 拷贝构造函数的调用时机

Test t0(1,2);
Test t1(1,2);
t1 = t0;//用t0给t1赋值， 不会调用拷贝构造函数，需要重载等号操作
  1. Test t2 = t0;
  2. Test t2(t0);
  3. 实参初始化形参的时候会调用类的拷贝构造函数
  4. 函数返回值是一个元素（复杂类型），返回的是一个新的匿名对象，所以调用匿名对象类的拷贝构造函数
void func1(){
    func2();//3.1 匿名对象并未被使用，接着调用其析构函数
    Test t = func2();//3.2 匿名对象用于初始化t，直接将匿名对象转成t，匿名对象不会被析构，也不会再调用拷贝构造函数
    Test t1(1,2);
    t1 = func2();//3.3 调用类的等号操作，匿名对象被析构
}
Test func2() {
    Test T;
    return T;//1.用T创建一个匿名对象，执行拷贝构造函数 2.析构T
}
void mian(){
    func1();
}

1.4 默认构造函数

  1. 默认无参构造函数：当类中没有定义构造函数时，编译器提供一个无参构造函数，函数体为空
  2. 默认拷贝构造函数：当类中没有定义拷贝构造函数时，编译器提供一个默认拷贝构造函数，进行成员变量的值复制（浅拷贝）

1.5 构造函数调用规则

  1. 当类中没有定义任何一个构造函数时，编译器会提供一个默认无参构造函数和默认拷贝构造函数
  2. 当类中定义了拷贝构造函数时，编译器不会提供默认无参构造函数
  3. 当类中定义了无参构造函数或有参构造函数，编译器不会提供默认无参构造函数

　　只要自己写了构造函数，就必须要用！

1.6 浅拷贝

class Test {
public:
    Test(const char *p){
        len_ = strlen(p);
        p_ = (char*)malloc(len_+1); //1
        strcpy(p_, p); //2
    }
    ~Test(){
        if(p_ != NULL){
            free(p_);
            p_ = NULL;
            len_ = 0;
        }
    }
private:
    char *p_;
    int len_;
};

void fun(){
    Test t1("abcdefg");
    Test t2 = t1;//3. 默认的拷贝构造函数，拷贝的是指针变量的值
}

void main() {
    fun();
}

　　上面的代码运行会崩溃，下面进行分析：

class Test {
public:
    Test(const char *p){
        len_ = strlen(p);
        p_ = (char*)malloc(len_+1); //1
        strcpy(p_, p); //2
    }
    Test(const Test& t){ //深拷贝
        len_ = t.len_;
        p_ = (char*)malloc(len_+1);
        strcpy(p_, t.p_);
    }
    ~Test(){
        if(p_ != NULL){
            free(p_);
            p_ = NULL;
            len_ = 0;
        }
    }
private:
    char *p_;
    int len_;
};

void fun(){
    Test t1("abcdefg");
    Test t2 = t1;//3. 自己写的拷贝构造函数，深拷贝
}

void main() {
    fun();
}

　　注意默认的等号操作也是浅拷贝！！！需要显示重载等号操作符。

1.7 构造函数初始化列表

class A{
public:
    A(int a) {
        a_ = a;
    }
private:
    int a_;
};

class B{
public:
    B(int b1, int b2){
        b1_ = b1;
        b2_ = b2;
    }
private:
    int b1_;
    int b2_;
    A a1_;
    A a2_;
};

void main(){
    A a1(1);
    B b1;
}

　　编译上面的代码会报错：没有合适的默认构造函数可用。在B类里面有A类型的成员变量，定义B类对象的时候，编译器要计算需要给B类对象分配多少内存，但是无法初始化a1_和a_2，因为A类没有默认的无参构造函数。

　　C++提供初始化列表对成员变量进行初始化。

　　拷贝构造函数一样的道理.

class A{
public:
    A(int a) {
        a_ = a;
    }
    ~A() {}
private:
    int a_;
};

class B{
public:
    B(int b1, int b2) : a1_(1), a2_(2){//1
        b1_ = b1;
        b2_ = b2;
    }
    B(int b1, int b2, int m, int n) : a1_(m), a2_(n){//2
        b1_ = b1;
        b2_ = b2;
    }
    ~B() {}
private:
    int b1_;
    int b2_;
    A a1_;
    A a2_;
};

void main(){
    A a1(1);
    B b1(1,2);//1
    B b2(1,2,3,4);//2
}

1.8 构造函数和析构函数的调用顺序

　　构造函数调用顺序：先执行A类对象的构造函数，如果有多个组合对象，按照定义顺序而不是按照初始化列表的顺序，即先进行a1_的构造，再进行a2_的构造，最后进行B类对象(b1或b2)的构造。

　　析构函数调用顺序：与构造函数调用顺序相反。

1.9 匿名对象的生命周期

class Test{
public:
    Test(int t){
        t_ = t;
        printf("contrust\n");
    }
    ~Test() {
        printf("decontrust\n")
    }
private:
    int t_;
};

void fun() {
    printf("start\n");
    Test(1);//1
    printf("end\n");
    printf("start\n");
    Test t = Test(1);//2
    printf("end\n");   
}

void mian(){
    fun();
}

　　执行上面程序会输出:

start contrust decontrust end start contrust end decontrust

  1. 先调用了Test构造函数,紧接着直接调用了析构函数,因为匿名对象没有其它对象去接,所以直接被析构.
  2. 匿名对象直接转化为t,不再调用拷贝构造函数

1.10 构造中调用构造(危险行为)

class Test{
public:
    Test(int a, int b){
        a_ = a;
        b_ = b;
    }
    Test(int a){
        a_ = a;
        Test(a, 2); //想把2初始化给b_
    }
    ~Test() {}
    int getB() const{
        return b_;
    }
private:
    int a_;
    int b_;
};

void main(){
    Test t(1);
    printf("b_: %d", t.getB());//输出多少?
}

　　执行上面程序发现输出b_为乱码,因为调用Test(a,2)会产生匿名对象,没有对象接它,直接析构,匿名对象和t对象没有一点儿关系.

2 析构函数

　　进行一些垃圾回收工作。

2.1 定义与调用

  1. 函数名与类名相同，且函数名前面加一个波浪号
  2. 用来清理对象的特殊成员函数
  3. 没有参数也没有任何返回类型的声明
  4. 对象销毁时自动调用
  5. 析构时先构造的对象后释放
