# 抽象数据类型
类的基本思想是**数据抽象**和**封装**，数据抽象是一种依赖于**接口**和**实现**分离的编程技术，接口包括用户所能执行的操作，实现包括类的数据成员、负责接口实现的函数体以及定义类所需要的各种私有函数；封装后的类隐藏了实现细节，类的用户只能使用接口而无法访问实现。
